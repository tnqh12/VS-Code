### 개인 자바스크립트 과제

# 아래 3가지 프로그램 중 원하는 1가지 프로그램을 구현
# 기간 : 6/20 ~ 6/21
# 발표 : 6/24 1교시

# 프로그램 1 : 퀴즈프로그램 - 이수보, 마윤주, 이용재
#             (jsquiz.html, jsquiz.js. jsquiz.css)
# 개요
# - 프로그램이 시작되면 이름을 입력 받는다.
# - 초기화면에서 이름과 점수를 고득점자 순으로 출력한다.
# - 10개의 단답형 퀴즈를 10초마다 한 문제씩 출제한다.
# - 건너뛰기 버튼을 클릭하면 다음퀴즈로 넘어간다.
# - 10개의 퀴즈를 끝내면 이름과 점수를 localStorage에 저장한다.

# 프로그램 2 : 레이싱프로그램 - 김민주, 진희준, 주민성, 최무진, 최원
#             (jsracing.html, jsracing.js. jsracing.css)
# 개요
# - 프로그램이 시작되면 5명의 주자(말, 자동차 ...)가 출발선에 있다.
# - START버튼을 누르면 5명의 주자가 결승선을 향해 이동한다.
# - 각 주자의 속도는 레이싱하는 동안 랜덤하게 변화하여야 한다.
# - 마지막 주자가 결승선에 도착하면 도착한 순서대로 등수와 소요시간을
#   화면에 표시한다.

# 프로그램 3 : 두더지게임프로그램 - 이형석, 정재훈, 박민서
#             (jsmole.html, jsmole.js. jsmole.css)
# 개요
# - 게임 초기화면에 최고점수를 달성한 사람의 이름과 점수가 표시된다.
# - 프로그램 시작시 9칸에 두더지가 한마리씩 들어 있다.
# - 60초의 기간동안 랜덤한 시간 간격으로 두더지가 한마리씩 머리를 내민다.
# - 마우스로 내민 머리를 클릭하면 점수가 올라간다.
# - 게임이 끝나면 이름과 점수를 localStorage에 저장한다.
# - (선택) 게임 난이도를 상/중/하로 선택하여 게임을 시작할 수 있다.
# - (선택) 슈퍼두더지 기능 : 빠른속도로 나왔다 들어감, 배점 높다.
